## 실행 방법

의존성 설치

```bash
yarn install
```

각 스크립트 실행

```bash
yarn start:answer1 # 문제 1번 실행
yarn start:answer2 # 문제 2번 실행
yarn start:answer3 # 문제 3번 실행
```
